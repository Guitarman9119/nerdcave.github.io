HTML, CSS and Sass

What are HTML and CSS?

HTML and CSS are the foundation of any web application. It is essential to know these two in web development before starting with any application. With HTML and CSS, you can still build static websites. An example of this could be a simple portfolio page, a small business site, an informational site, etc.

Hypertext markup language (HTML) is the standard markup language for creating Web pages. HTML is not a programming language as it does not contain any programming logic. HTML describes the structure of a Web page and is extremely important. HTML describes various document (Web page) such as headings, paragraphs, and lists. You can see HTML as the underlying structure of your website. HTML is the starting point of any website either by writing it out or generated by text editors. HTML on its display content but has no style and looks very basic.

Cascading Style Sheets (CSS) comes to the rescue to give our HTML style. CSS describes how the content of our web page is displayed. The user sees the page: fonts, colors, background, images, line spacing, page layout, etc. CSS allows us to change how these content are displayed CSS also has precompilers such as Sass/Less/Stylus. This is known as a preprocessor scripting language that is interpreted or compiled into Cascading Style sheets. We will cover Sass in-depth in another section.
